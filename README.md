## Set up

1. リポジトリをクローンします

```bash
git clone
```

2. 仮想環境を作成します

```bash
python3 -m venv venv
```

3. 仮想環境を有効化します

```bash
source venv/bin/activate
```

4. 依存関係をインストールします

```bash
pip install -r requirements.txt
```

5. プログラムを実行します

```bash
python3 main.py
```
